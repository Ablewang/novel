# 大纲模版：规则怪谈流 (脑力求生)

**核心卖点：** 诡异规则 + 逻辑推理 + 惊悚氛围 + 直播弹幕

---

## 一、 核心设定 (填空)

*   **怪谈降临机制：** (例：全球随机抽取天选者 / 惊悚游戏APP强制拉人)
*   **主角的金手指：** (只能选一个，太强就不好看了)
    *   [ ] **死亡回档** (S/L大法，试错)
    *   [ ] **提示系统** (能看到隐藏的正确/错误提示)
    *   [ ] **天赋：叛逆** (违反规则反而变强)
*   **第一个副本场景：** (例：妈妈的纸条 / 诡异学校 / 最后一班地铁)

---

## 二、 剧情填空 (单副本节奏)

> **1. 规则降临 (开局)：**
> *   主角进入副本，拿到了一张**【规则纸条】**。
> *   *第一条规则是：* (例：不要给陌生人开门)
> *   *最后一条规则是：* (例：如果看到红衣人，立刻自杀)
>
> **2. 第一次危机 (试错)：**
> *   诡异出现，违反了规则A。
> *   其他国家/队伍的选手**直接暴毙** (全球直播震惊)。
> *   主角利用金手指发现了规则A的**漏洞/错误**，成功存活。
>
> **3. 探索与推理 (压抑)：**
> *   主角发现这地方不对劲，原来的规则之间**互相矛盾**。
> *   (例：规则1说这世上没有鬼，规则5说遇到鬼要闭眼。)
> *   主角必须推导出**“隐藏的世界观真相”**。
>
> **4. 最终通关 (高潮)：**
> *   所有人都绝望了，怪谈彻底爆发。
> *   主角做出了一个**看似找死**的行为 (其实是卡BUG)。
> *   副本崩塌 / 诡异被收容 / 完美通关。
>
> **5. 结算与影响 (爽点)：**
> *   主角获得评分 SSS。
> *   主角所在的国家获得资源奖励 / 免除灾难。
> *   全网沸腾。

---

## 三、 常用规则套路 (抄作业)
*   **认知污染类**：看到的东西不一定是真的 (如果你看到月亮变红，请立刻吃药)。
*   **身份扮演类**：你必须扮演好你的角色 (你是医生，绝对不能拒绝病人)。
*   **矛盾对立类**：蓝衣员工是好的，红衣员工是坏的。
