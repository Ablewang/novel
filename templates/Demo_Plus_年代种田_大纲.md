# 大纲模版：年代/四合院流 (怀旧致富)

**核心卖点：** 时代情怀 + 斗极品亲戚 + 商业发家 + 弥补遗憾

---

## 一、 核心设定 (填空)

*   **穿越时间点：** (例：1980年改革开放前夕 / 1960年饥荒时期)
*   **主角身份：** (例：下乡知青 / 街溜子 / 四合院里的普通住户)
*   **家庭背景 (必须有极品)：**
    *   [ ] 偏心的父母 (只爱大哥/弟弟)
    *   [ ] 吸血的亲戚 (借钱不还)
    *   [ ] 势利的邻居 (看主角穷就踩一脚)
*   **金手指：** (不要太夸张)
    *   [ ] **随身空间** (带个超市/仓库)
    *   [ ] **灵泉水** (种地养殖神速)
    *   [ ] **签到系统** (每天给点肉票/粮票)

---

## 二、 剧情填空 (致富节奏)

> **1. 开局 (一地鸡毛)：**
> *   主角醒来，发现家徒四壁，老婆/妹妹饿得面黄肌瘦。
> *   极品亲戚/邻居上门找茬 (要房/要钱)。
> *   主角**当场翻脸/分家**，不再做包子，展现强硬一面。
>
> **2. 第一桶金 (投机倒把)：**
> *   利用金手指或超前眼光，搞到了第一批紧俏物资 (肉/电子表/衣服)。
> *   偷偷去黑市/摆摊，赚到了普通人一年的工资。
> *   **打脸环节：** 买肉/买新衣回家，馋死隔壁邻居。
>
> **3. 事业起步 (风口)：**
> *   承包土地 / 开饭馆 / 倒腾国库券。
> *   收服几个靠谱的小弟。
> *   解决眼红来捣乱的流氓/竞争对手。
>
> **4. 情感升温 (宠妻/娃)：**
> *   给老婆买雪花膏，给孩子买大白兔奶糖。
> *   弥补前世对家人的亏欠。
>
> **5. 结局 (首富)：**
> *   从个体户变成大企业家。
> *   曾经的极品亲戚现在跪着求主角施舍。

---

## 三、 常用爽点元素
*   **吃肉**：在那个缺衣少食的年代，大口吃红烧肉就是最顶级的炫富。
*   **大件**：买自行车、缝纫机、电视机，全院轰动。
*   **高考**：恢复高考，主角一鸣惊人，考上北大清华。
