# 大纲模版：苟道/长生流 (反内卷)

**核心卖点：** 稳健发育 + 熬死仇人 + 扮猪吃虎 + 岁月沧桑

---

## 一、 核心设定 (填空)

*   **主角性格：** 极度怕死，被迫害妄想症，凡事要做最坏的打算。
*   **金手指：** (必须与寿命/发育有关)
    *   [ ] **长生不老** (寿命无限，资质极差)
    *   [ ] **熟练度面板** (只要练就能变强，无瓶颈)
    *   [ ] **属性加点** (每天加一点属性)
*   **身份伪装：** (例：天牢狱卒 / 宗门杂役 / 图书馆管理员)

---

## 二、 剧情填空 (时间跨度)

> **1. 开局 (定调)：**
> *   主角穿越，发现修仙界太危险（杀人夺宝）。
> *   决定**隐藏实力**，找个安全的地方混日子。
> *   立下规矩：不凑热闹、不争宝物、不沾因果。
>
> **2. 冲突 (送走一代)：**
> *   当年的天才/仇人风光无限，嘲笑主角是缩头乌龟。
> *   几十年后，天才战死/老死，**主角依然年轻**。
> *   主角在仇人坟头倒了一杯酒：“道友，你输在命短。”
>
> **3. 危机 (不得不出手)：**
> *   躲藏的地方被波及（宗门被灭/王朝更替）。
> *   主角被迫出手，**一招秒杀**来犯之敌（因为偷偷练了几百年）。
> *   **事了拂衣去，深藏功与名**，换个地方继续苟。
>
> **4. 循环 (岁月史书)：**
> *   看着曾经救过的小女孩变成了一代女帝。
> *   看着种下的树变成了通天神木。
> *   主角成为传说中的“幕后黑手/老祖宗”。
>
> **5. 结局 (无敌)：**
> *   终于有一天，主角发现自己已经无敌于世间。
> *   出山，镇压一切不服。

---

## 三、 常用苟道法则 (金句)
*   “君子报仇，十年太晚。我是小人，我一万年都不晚。”
*   “只要我活得够久，你们都是历史的尘埃。”
*   “凡事留亿手底牌。”
