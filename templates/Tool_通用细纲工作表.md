### 4. 写前检查 (Pre-Check)

*   [ ] 本章有没有**废话**？*(大段环境描写、无关路人对话 → 删)*
*   [ ] 主角有没有**吃瘪过夜**？*(爽文严禁，其他类型酌情)*
*   [ ] 结尾钩子够不够**勾人**？*(不够就加"突然……")*
*   [ ] 信息量是否**过载**？*(一章别解决超过2个问题)*
*   [ ] 节奏是否**拖沓**？*(能500字说完的别用1000字)*

## 附：章节类型速查 (终极扩展版)

| 章节类型 | 核心情绪 | 推荐情节组合 (Combo) |
| :--- | :--- | :--- |
| **💥 爽文·打脸章** | 愤怒 → 释放 | **路人嘲讽/反派跳脚** → 主角忍耐/蓄力 → **一招反杀/身份曝光** |
| **✨ 爽文·装逼章** | 质疑 → 震惊 | **众人看衰/难度极高** → 主角淡定出手 → **全场寂静/疯狂脑补** |
| **🎁 爽文·收获章** | 期待 → 满足 | **付出代价/战胜强敌** → 获得宝物/技能 → **测试威力/震惊队友** |
| **❤️ 言情·撒糖章** | 暧昧 → 姨母笑 | **意外接触/共同危机** → 肢体接触/眼神拉丝 → **心动瞬间/确立关系** |
| **💔 言情·修罗场** | 紧张 → 胃疼 | **甜蜜互动** → 第三者(前任/情敌)闯入 → **极限拉扯/二选一** |
| **🧠 智斗·布局章** | 困惑 → 恍然 | **身陷绝境/看似无解** → 对手得意洋洋 → **逻辑反转/底牌揭晓** |
| **😱 悬疑·惊悚章** | 不安 → 恐惧 | **异常现象/诡异规则** → 尝试探索失败 → **未知逼近/生死一线** |
| **🍻 日常·团建章** | 放松 → 温情 | **战后聚餐/庆功宴** → 搞笑插曲/醉酒互动 → **加深羁绊/伏笔暗埋** |
| **🆙 成长·突破章** | 压抑 → 升华 | **瓶颈困扰/心魔滋生** → 顿悟/获得关键指点 → **突破异象/实力暴涨** |
| **🔍 剧情·填坑章** | 震撼 → 满足 | **所有线索汇聚** → 揭开惊天真相 → **世界观颠覆/旧坑填平+新坑开启** |
