# 大纲模版：国运/直播流 (集体荣誉)

**核心卖点：** 全球竞赛 + 国家绑定 + 弹幕互动 + 资源具现

---

## 一、 核心设定 (填空)

*   **危机背景：** (例：资源枯竭 / 异界入侵 / 神明游戏)
*   **规则：** 每个国家随机抽取一名选手进入禁地。选手获得的资源，**百倍具现**给现实中的国民。选手死亡，国家降临灾难。
*   **主角人设：** (必须有反差，让观众一开始看衰)
    *   [ ] **扮演流**：扮演盲人剑客/酒剑仙/疯子。
    *   [ ] **看似弱鸡**：穿着拖鞋短裤的宅男。
*   **队友 (通常双人组)：** 一个高冷强大的花瓶队友 (如白月魁)。

---

## 二、 剧情填空 (直播节奏)

> **1. 选手选拔 (欲扬先抑)：**
> *   名单公布。别国选手全是特种兵/贝爷。
> *   龙国选手（主角）是个**只会喝酒的醉鬼/瞎子**。
> *   **弹幕反应：** “完了！天亡我龙国！” “能不能换人啊！”
> *   外国网友嘲讽：“龙国这次要灭国了。”
>
> **2. 禁地首战 (第一次打脸)：**
> *   遭遇恐怖凶兽。别国选手惨死。
> *   主角**拔刀/睁眼**，一击秒杀凶兽。
> *   **弹幕反应：** “卧槽！大佬！！” “对不起，我刚才说话声音大了点！”
>
> **3. 资源具现 (集体爽点)：**
> *   主角发现一片**绿洲/石油/灵矿**。
> *   【公告：龙国选手占领绿洲，百倍具现于龙国！】
> *   现实中：龙国沙漠变绿洲，全民寿命+10。
> *   外国网友：羡慕嫉妒恨，跪求移民。
>
> **4. 危机升级 (国战)：**
> *   其他国家选手联合起来想杀主角。
> *   主角一打多，团灭对方。
> *   对方国家降临天灾（惩罚）。

---

## 三、 常用弹幕套路
*   **绝望风**：“完了完了，这次真完了。”
*   **震惊风**：“奈何本人没文化，一句卧槽行天下。”
*   **凡尔赛风**：“哎，这就是躺赢的感觉吗？别的国家还在吃草，我们在吃肉。”
