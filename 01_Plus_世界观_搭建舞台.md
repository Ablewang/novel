# 01_Plus_世界观：搭建舞台 (小白防卡文版)

在大纲和人物都有了之后，别急着写正文。
你需要先搭个台子，否则写的时候会因为**“起名字”**和**“想设定”**卡死。

本指南只包含**最核心、必须有**的设定。别搞太复杂，够用就行。

---

## 一、 战力体系 (谁强谁弱？)

读者最关心的就是“主角现在有多强”。你必须给一个明确的**等级梯队**。

### 🔢 填空模版 (建议 5-9 个等级)
*   **Lv.1 (入门/炮灰)：** ________ (例：练气 / 见习 / 下忍) -> *主角开局等级*
*   **Lv.2 (精英/小怪)：** ________ (例：筑基 / 正式 / 中忍)
*   **Lv.3 (高手/小Boss)：** ________ (例：金丹 / 资深 / 上忍)
*   **Lv.4 (大佬/城主)：** ________ (例：元婴 / 大师 / 影)
*   **Lv.5 (传说/天花板)：** ________ (例：化神 / 宗师 / 六道)

> **💡 秘诀：** 如果起名废，直接抄经典的（青铜/白银/黄金，或者 F/E/D/C/B/A/S）。**千万别自创生僻字等级！**

---

## 二、 货币与物价 (钱从哪来？)

很多小白写着写着就崩了，因为**物价乱了**。
先定好“购买力锚点”。

### 💰 设定填空
1.  **货币名称：** ________ (例：灵石 / 金魂币 / 联邦点 / 只有数字)
2.  **购买力锚点 (1个货币能买啥？)：**
    *   *低端锚点：* 1 块钱 = 一个馒头 / 一瓶水。
    *   *高端锚点：* 10000 块钱 = 一把入门武器 / 一本基础技能书。
3.  **主角现在的资产：** ________ (建议设为 0 或 负债，制造缺钱的焦虑感)

---

## 三、 地图与势力 (去哪打怪？)

不要画世界地图！只需要**“新手村”**和**“第一张地图”**。

### 🗺 势力填空
1.  **新手村 (主角老家)：** ________ (例：乌坦城 / 第七区 / 羊村)
    *   *这里谁说了算？* (反派势力)：________
    *   *主角的靠山/庇护所？*：________
2.  **远方的大城市 (主角向往的地方)：** ________ (例：云岚宗 / 帝都 / 伟大航路)

---

## 四、 核心冲突 (大家在抢什么？)

这个世界的人，终其一生都在争夺什么资源？

*   **[ ] 抢灵气/资源** (修仙/末世文)
*   **[ ] 抢地盘/生存空间** (争霸/种田文)
*   **[ ] 抢权柄/神位** (诡秘/西幻文)
*   **[ ] 抢钱/名声** (都市/娱乐文)

> **💡 作用：** 当你不知道配角为什么要打架时，就让他们为了这个东西打。
