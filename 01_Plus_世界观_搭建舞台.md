# 01_Plus_世界观：搭建舞台 (小白防卡文版)

在大纲和人物都有了之后，别急着写正文。
你需要先搭个台子，否则写的时候会因为**“起名字”**和**“想设定”**卡死。

本指南只包含**最核心、必须有**的设定。别搞太复杂，够用就行。

## 一、 战力体系 (谁强谁弱？)

读者最关心的就是“主角现在有多强”。你必须给一个明确的**等级梯队**。

### 🔢 填空模版 (建议 5-9 个等级)
*   **Lv.1 (入门/炮灰)：** ________ (例：练气 / 见习 / 下忍) -> *主角开局等级*
*   **Lv.2 (精英/小怪)：** ________ (例：筑基 / 正式 / 中忍)
*   **Lv.3 (高手/小Boss)：** ________ (例：金丹 / 资深 / 上忍)
*   **Lv.4 (大佬/城主)：** ________ (例：元婴 / 大师 / 影)
*   **Lv.5 (传说/天花板)：** ________ (例：化神 / 宗师 / 六道)

> **💡 秘诀：** 如果起名废，直接抄经典的（青铜/白银/黄金，或者 F/E/D/C/B/A/S）。**千万别自创生僻字等级！**

## 二、 货币与物价 (钱从哪来？)

很多小白写着写着就崩了，因为**物价乱了**。
先定好“购买力锚点”。

### 💰 设定填空
1.  **货币名称：** ________ (例：灵石 / 金魂币 / 联邦点 / 只有数字)
2.  **购买力锚点 (1个货币能买啥？)：**
    *   *低端锚点：* 1 块钱 = 一个馒头 / 一瓶水。
    *   *高端锚点：* 10000 块钱 = 一把入门武器 / 一本基础技能书。
3.  **主角现在的资产：** ________ (建议设为 0 或 负债，制造缺钱的焦虑感)

## 五、 进阶：世界观与金手指的黑盒理论 (渐进式设定)

很多新手会陷入一个误区：*“我必须在一开始就把金手指的来源、原理、100级后的功能全部设定好，否则就是不严谨。”*
**大错特错！** 这样做只会让你无法动笔。

### 1. 黑盒理论
对于读者来说，金手指/系统是一个**黑盒**。
*   **Input (输入)：** 怎么用？ (例：摸一下尸体)
*   **Output (输出)：** 有什么爽点？ (例：获得一个超强词条)
*   **Process (原理)：** **不需要解释！** (至少前50万字不需要)

### 2. 渐进式设定法 (洋葱剥皮法)
设定是随着剧情生长出来的，不是一开始就盖好的楼。

| 阶段 | 剧情进度 | 设定重点 | 案例 (系统流) |
| :--- | :--- | :--- | :--- |
| **第一层** | **开篇前3章** | **UI与交互** | 只能看到词条，摸尸体能提取。 (让读者秒懂怎么爽) |
| **第二层** | **前20章** | **代价与边界** | 发现融合金色词条会消耗精神力；不能融合相斥属性。 (增加战术深度) |
| **第三层** | **中期 (50-100章)** | **版本更新** | 主角筑基了，系统解锁【合成台】功能。 (解决审美疲劳) |
| **第四层** | **后期/大结局** | **本源真相** | 系统其实是天道的碎片。 (升华主题，填坑) |

### 3. 如何确保前后不矛盾？ (防崩坏守则)
不敢随便补设定，是怕打脸？掌握这三招，想怎么补就怎么补。

*   **A. 借用“认知局限” (最常用)**
    *   *前期：* 主角以为这是个武侠系统。
    *   *后期补丁：* 主角接触修仙者后恍然大悟——“原来那不是内力，是灵气！是我之前见识太短浅了！” (设定升级了，逻辑没崩，因为是主角看错了)。
*   **B. 话别说死 (留后路)**
    *   不要写：“系统**绝对**无法升级。”
    *   要写：“主角**目前**发现，系统**似乎**没有升级的迹象。”
*   **C. 后补伏笔法 (时空穿越)**
    *   突然想加个“水下呼吸”的功能？
    *   *操作：* 利用电子文档的优势，回到第10章主角获得水系宝物时，加一句：“当时他感觉肺部一阵清凉，但他没在意。”
    *   *现在：* 激活隐藏被动！读者还会夸你草蛇灰线。
