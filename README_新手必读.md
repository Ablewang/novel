## 🚀 你的第一天行动指南 (Day 1)

别想以后几百万字的事，今天只做这三件事：

1.  **打开 `01_准备阶段_素材与大纲.md`**，选一个喜欢的流派模板，把那5个空填了。(耗时：10分钟)
2.  **打开 `02_角色塑造_捏人指南.md`**，给主角起个名字，定个金手指。(耗时：5分钟)
3.  **打开 `03_实战过程_边写边修.md`**，看一眼“黄金开篇三板斧”，然后在你的文档里写下第一句话。(耗时：1分钟)

**总计耗时：16分钟。** 做完这些，你就是个小说作者了！加油！
