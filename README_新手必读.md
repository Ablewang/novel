## 🚀 你的第一天行动指南 (Day 1)

1.  打开 **`templates/`** 文件夹，把里面的大纲文件名看一遍。
2.  选一个你最感兴趣的类型 (比如 *规则怪谈*)。
3.  复制那个文件，把里面的空填上。
4.  你的小说架构就搭建好了！
