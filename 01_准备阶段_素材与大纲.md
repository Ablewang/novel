# 01_准备阶段：素材与大纲 (极速爽文版)

本指南针对**当代快节奏阅读习惯**进行了优化。
如果你的目标是让读者**“点开就不想退出去”**，请用这个版本的大纲公式。

---

## 一、 核心脑洞 (一句话卖点)

在写大纲前，先问自己：**我的书凭什么让读者点进来？**
现在的读者看书先看**书名**和**简介**。

### ⚡️ 脑洞自检公式：
> **[ 极度的反差/猎奇 ] + [ 极致的利益/爽点 ]**

*   *普通脑洞：* 一个少年努力修炼成仙。（太土了，没人看）
*   *极速脑洞：* **呼吸就能变强？** 我在修仙界卷死所有天才！
*   *极速脑洞：* 全球灾变，**只有我拥有安全屋**，女神跪求我开门。

---

## 二、 宏观大纲：新媒体流派填空 (爽点前置版)

这里对经典流派进行了**“提速”**，删去了压抑的环节。

### ⚔️ 类型 A：玄幻 / 高武 (无敌流)
> **1. 开局暴击 (前3章)：**
> *   主角遭遇生死危机/退婚羞辱，**系统/金手指立刻到账**，当场反杀/打脸，绝不隔夜。
>
> **2. 期待感 (主线)：**
> *   利用金手指的**信息差/资源差**，一路碾压。
> *   *核心爽点：* 别人拼命抢的宝物，主角当垃圾扔；别人练不成的神功，主角看一眼就会。
>
> **3. 节奏循环：**
> *   换地图 -> 被看不起 -> 展示实力 -> 震惊全场 -> 收获资源 -> 换下一个地图。

### 🏰 类型 B：都市 / 反转 (马甲流)
> **1. 开局暴击：**
> *   主角身份低微（赘婿/送外卖），被权贵羞辱。
> *   **反转：** 一个电话打来，主角其实是世界首富/战神/神医，手下立刻开直升机来接。
>
> **2. 期待感 (主线)：**
> *   隐瞒身份装逼。看着反派像小丑一样跳，最后亮出身份吓死他们。
>
> **3. 节奏循环：**
> *   反派挑衅 -> 主角忍让(装弱) -> 反派得寸进尺 -> 主角亮身份/打电话 -> 反派跪地求饶。

### 🌾 类型 C：种田 / 囤货 (安全感流)
> **1. 开局暴击：**
> *   末世马上降临/穿越到荒岛。主角获得**无限空间/百倍增幅**。
> *   疯狂囤积物资，要在灾难来临前做好准备。
>
> **2. 期待感 (主线)：**
> *   **对比爽**：外面的人饿得吃土，主角在屋里吃火锅、喝冰可乐。
>
> **3. 节奏循环：**
> *   缺某种资源 -> 外出探索/交易 -> 遇到眼红的掠夺者 -> 轻松反杀 -> 获得更多资源升级基地。

## 三、 微观章纲 (情绪过山车)

每一章都要有钩子。

> **1. 开头 (黄金3秒)：**
> *   必须承接上一章的悬念，或者直接抛出新冲突。**不要写景！**
>
> **2. 中间 (情绪拉扯)：**
> *   反派嘲讽要**狠**（拉仇恨）。
> *   路人震惊要**真**（做气氛组）。
>
> **3. 结尾 (钩子)：**
> *   **断章技巧**：剧情卡在最高潮的地方。
>     *   *例：* 主角举起了刀，对着反派的脖子砍了下去——(本章完)。
>     *   *例：* 就在这时，门外传来了一个意想不到的声音——(本章完)。

## 四、 进阶：如何创造新流派 (缝合怪法)

公式不变，但**元素要更新**。

*   **旧缝合：** 历史 + 穿越。
*   **新缝合：** 历史 + **直播弹幕** + **国运**。
    *   *《国运大秦》*：秦始皇的一举一动，决定了现代龙国的国运。全球观众都在看。
