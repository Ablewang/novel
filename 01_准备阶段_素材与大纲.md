# 01_准备阶段：素材与大纲 (极速爽文版)

本指南针对**当代快节奏阅读习惯**进行了优化。
如果你的目标是让读者**“点开就不想退出去”**，请用这个版本的大纲公式。

---

## 一、 核心脑洞 (一句话卖点)

在写大纲前，先问自己：**我的书凭什么让读者点进来？**
现在的读者看书先看**书名**和**简介**。

### ⚡️ 脑洞自检公式：
> **[ 极度的反差/猎奇 ] + [ 极致的利益/爽点 ]**

*   *普通脑洞：* 一个少年努力修炼成仙。（太土了，没人看）
*   *极速脑洞：* **呼吸就能变强？** 我在修仙界卷死所有天才！
*   *极速脑洞：* 全球灾变，**只有我拥有安全屋**，女神跪求我开门。

---

## 二、 素材挖掘：万物皆可爽文 (转化公式)

灵感不是等来的，是**“抄”**来的（从现实中抄）。
只有把碎片化的素材转化为**剧情**，大纲才是有血有肉的。

### 🔍 1. 高效素材源
去哪里找那些让读者**血压升高**（然后期待打脸）的素材？
*   **知乎/微博热搜/贴吧：** 专门看**矛盾点**。婆媳矛盾、职场压榨、奇葩相亲、熊孩子毁手办。
    *   *转化思路：* 把现实中的憋屈，在小说里用金手指**百倍奉还**。
*   **短视频评论区：** 专门看**“意难平”**。
    *   *转化思路：* 视频里好人没好报？你的书里让他好人成神，坏人下地狱。
*   **奇葩新闻：** 现实往往比小说更离谱。

### 🧪 2. 素材转化公式
不要照搬新闻，要**加料**。

> **公式：[ 憋屈的现实素材 ] + [ 不讲理的金手指 ] = [ 极致爽文剧情 ]**

*   *素材（现实）：* 公司裁员，35岁老员工被暴力辞退，没赔偿。（憋屈，令人愤怒）
*   *加料（网文）：* 主角获得“神豪系统”/“收购卡”。
*   *剧情（爽点）：*
    1.  主管嚣张：你被开除了，滚！
    2.  主角淡定掏手机操作。
    3.  反转：大老板冲进来给主角敬茶。主角：现在你被开除了。

### 📂 3. 剧情积木搭建
把收集的素材分类，写大纲时像**搭积木**一样填进去，不用现想。
*   **打脸库：** 收集10种看不起主角的方式（势利眼亲戚、绿茶前任、狗眼看人低的服务员）。
*   **震惊库：** 收集10种路人震惊的反应（眼珠瞪出来、下巴掉地上、怀疑人生）。
*   **宝物库：** 各种开挂道具的功能（透视、读心、时间停止）。

## 三、 宏观大纲：新媒体流派填空 (爽点前置版)

这里对经典流派进行了**“提速”**，删去了压抑的环节。

### ⚔️ 类型 A：玄幻 / 高武 (无敌流)
> **1. 开局暴击 (前3章)：**
> *   主角遭遇生死危机/退婚羞辱，**系统/金手指立刻到账**，当场反杀/打脸，绝不隔夜。
>
> **2. 期待感 (主线)：**
> *   利用金手指的**信息差/资源差**，一路碾压。
> *   *核心爽点：* 别人拼命抢的宝物，主角当垃圾扔；别人练不成的神功，主角看一眼就会。
>
> **3. 节奏循环：**
> *   换地图 -> 被看不起 -> 展示实力 -> 震惊全场 -> 收获资源 -> 换下一个地图。

### 🏰 类型 B：都市 / 反转 (马甲流)
> **1. 开局暴击：**
> *   主角身份低微（赘婿/送外卖），被权贵羞辱。
> *   **反转：** 一个电话打来，主角其实是世界首富/战神/神医，手下立刻开直升机来接。
>
> **2. 期待感 (主线)：**
> *   隐瞒身份装逼。看着反派像小丑一样跳，最后亮出身份吓死他们。
>
> **3. 节奏循环：**
> *   反派挑衅 -> 主角忍让(装弱) -> 反派得寸进尺 -> 主角亮身份/打电话 -> 反派跪地求饶。

### 🌾 类型 C：种田 / 囤货 (安全感流)
> **1. 开局暴击：**
> *   末世马上降临/穿越到荒岛。主角获得**无限空间/百倍增幅**。
> *   疯狂囤积物资，要在灾难来临前做好准备。
>
> **2. 期待感 (主线)：**
> *   **对比爽**：外面的人饿得吃土，主角在屋里吃火锅、喝冰可乐。
>
> **3. 节奏循环：**
> *   缺某种资源 -> 外出探索/交易 -> 遇到眼红的掠夺者 -> 轻松反杀 -> 获得更多资源升级基地。

## 四、 微观章纲 (情绪过山车)

每一章都要有钩子。

> **1. 开头 (黄金3秒)：**
> *   必须承接上一章的悬念，或者直接抛出新冲突。**不要写景！**
>
> **2. 中间 (情绪拉扯)：**
> *   反派嘲讽要**狠**（拉仇恨）。
> *   路人震惊要**真**（做气氛组）。
>
> **3. 结尾 (钩子)：**
> *   **断章技巧**：剧情卡在最高潮的地方。
>     *   *例：* 主角举起了刀，对着反派的脖子砍了下去——(本章完)。
>     *   *例：* 就在这时，门外传来了一个意想不到的声音——(本章完)。

---

## 五、 进阶：如何创造新流派 (缝合怪法)

公式不变，但**元素要更新**。

*   **旧缝合：** 历史 + 穿越。
*   **新缝合：** 历史 + **直播弹幕** + **国运**。
    *   *《国运大秦》*：秦始皇的一举一动，决定了现代龙国的国运。全球观众都在看。
